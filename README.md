## Project List

### Balloon Shooting Game
A game where the player shoots balloons using arrows. The game tracks the score, missed balloons, and allows the player to restart after game over.

### Calorie Counter
A program to calculate total calories, protein, carbs, and fat based on selected food items. Includes categories like fruits, vegetables, grains, and meat.

### Easypaisa
A simulation of an EasyPaisa application for balance checking, depositing, withdrawing, bill payments, ticket purchases, and food ordering.

### Encryption and Decryption System
Implements monoalphabetic substitution encryption and decryption. Allows users to encrypt and decrypt messages using predefined substitution tables.

### Medical Billing System
A program for managing medical store transactions. Includes functionality for buying medicines, viewing statistics, and calculating total earnings.

### Parking Management System
A system to manage parking spaces for different vehicle types. Tracks the number of vehicles parked, calculates charges, and maintains records.

### Shoes Management System
A program for managing shoe inventory and sales. Tracks stock, calculates total sales, and displays statistics for different shoe categories.

## Contribute With Your Code!
***Please follow these guidelines when submitting a pull request:***
- Use the same file naming format. **Example:** _Project Name.asm_
- Add a README file with a detailed explanation of the implemented concept.
- If the project already exists, specify that this is another implementation in the file name. **Example:** _Project Name [Another Implementation].asm_
- Test your code thoroughly before submitting.
